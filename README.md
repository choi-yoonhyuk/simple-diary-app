# Simple Diary

> 5가지 감정을 골라 오늘의 하루를 표현하는 일기 앱 "Simple Diary"입니다.
> </br>

'완전 좋음', '좋음', '그러저럭', '나쁨', '끔찍함' 총 5가지 감정을 선택하여 오늘 나의 감정을 토대로 일기를 쓰는 서비스 입니다.

---

## 설치 방법

위치: root 폴더

```sh
npm install
```

---

## 세부 기능

1. Create

- 일기를 생성 하는 기능

2. Read

- 일기 세부정보를 확인할 수 있는 기능

3. Delete

- 일기를 삭제할 수 있는기능

4. Sort

- 최신순, 오래된 순으로 분류되는 기능
- 감정에 따라 분류되는 기능

5. List

- 날짜에 맞춰 일기리스트가 리스트업 기능

6. Update

- 일기 날짜 , 감정, 내용을 수정할 수 있는 기능
